# 🛒 Ecommerce Business Analytics Project

## 📌 Project Overview

This project focuses on analyzing ecommerce business performance using Python, SQL, and Power BI. The objective was to identify operational inefficiencies, revenue trends, delivery delays, order cancellations, and customer behavior patterns using real-world ecommerce data.

---

# 🎯 Business Problems Solved

- Delivery delay analysis
- Order cancellation root cause analysis
- Revenue and product performance analysis
- Customer retention and purchase behavior analysis
- Customer satisfaction impact analysis

---

# 🧰 Tech Stack

- Python
- Pandas
- NumPy
- SQL (MySQL)
- Power BI
- DAX

---

# 🔍 Key Insights

- Delivery delays negatively impacted customer review scores
- Certain product categories contributed disproportionately to cancellations
- Revenue was concentrated among a limited number of categories
- Customer satisfaction influenced repeat purchasing behavior
- Logistics inefficiencies affected operational performance

---

# 💡 Business Recommendations

- Improve logistics operations for high-delay categories
- Monitor cancellation-prone product segments
- Focus marketing on high-retention categories
- Enhance delivery efficiency to improve customer satisfaction
